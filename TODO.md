## 🚀 Deployment Steps

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Portfolio ready"
   git remote add origin YOUR_GITHUB_URL
   git push -u origin main
   ```

2. **Deploy on Vercel**
   - Go to vercel.com
   - Import your GitHub repo
   - Click Deploy
   - Done!
